Java: A Programming Language Meant for Frustration

By De Leon Kenxie Jermaine S. 16 January 2026
Introduction

Have you ever experienced typing several lines of code just for a simple word like Hello World to show up? Or making a simple loop in other programming languages feel overcomplicated in Java? Let me tell you about the frustration I dealt with while trying to learn Java.

# Python
print("Hello World")
// Java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
First Encounter and Frustrations

I started learning Java during my first year of college. At first, I thought it would be easier than Python because my brother told me, Java is a lot easier than Python. In hindsight, my brother lied to me. The number of rules and methods required to perform simple tasks in Java overwhelmed me. It took hours just to make simple loops work without errors.

Coming from Python made Java feel unnecessarily complex. I was also not in the right mindset at the time. Because when I learn and practice Java my mind keeps saying This could be a lot less complicated in Python or I would have done this faster in Python. Which did not help me learn or even appreciate Java.

Realizations and a Wake up call

I learned that my course is not just about learning. It is all about applying the things you learned and putting it into practice. Java was a harsh reality check for me and showed my weakness in my mindset when it comes to learning and applying things.

With how much I have changed since the last time I learned Java, maybe it's time to learn Java again with a fresh new mindset.
